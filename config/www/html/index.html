Built for Power

Every layer engineered for correctness, performance and flexibility.

⚡

Non-Blocking I/O

Full poll()-based event loop. Thousands of simultaneous connections, zero thread overhead.

📡

HTTP/1.1 Parser

Handles GET, POST and DELETE. Supports chunked transfer encoding, query strings, and custom headers.

⚙️

Config Engine

NGINX-inspired config syntax. Multiple virtual servers, location blocks, custom error pages and redirects.

📂

Static File Serving

Root directories, index files, directory autoindex and correct MIME type detection out of the box.

🐍

CGI Support

Execute Python, PHP or any CGI-compliant script. Configure per-location with cgi_ext and cgi_path.

📤

File Uploads

Per-location upload stores with configurable body size limits. Secure, isolated upload handling.
